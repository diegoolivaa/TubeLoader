# TubeLoader

Simple and minimalistic application made in Python for downloading YouTube videos 📽️

![Image](https://github.com/user-attachments/assets/4d687238-3e06-4fb9-beed-c263318bd0eb)

Tutorial:
-  Download the .zip file from the "Releases" tab ✅
-  Extract the zip folder ✅
-  Execute TubeLoader.exe ✅
-  Find your YouTube video ✅
-  Paste it in the program ✅
-  Select wether you'd want to download the video or the audio of the YouTube video ✅📽️
